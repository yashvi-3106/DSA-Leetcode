# 📝 Symmetric Binary Tree (LeetCode 101)

## 📌 Problem Statement  
Given a **binary tree**, check whether it is a **mirror of itself** (i.e., symmetric around its center).  

- A tree is symmetric if the left subtree is a **mirror reflection** of the right subtree.

---

## 🔍 Example  

**Input:**  
```
1
/
2 2
/ \ /
3 4 4 3
```

**Output:**  
true

**Input:**  
```
1
/
2 2
\
3 3
```

**Output:**  
false

---

## 🚀 Approaches  

### ✅ Method 1: Recursion (DFS)  
- Recursively compare **left and right subtrees**.  
- Check that:
  1. Values of current nodes are equal.  
  2. Left subtree of left node = Right subtree of right node.  
  3. Right subtree of left node = Left subtree of right node.  

**Time Complexity:** O(n) → Visit each node once  
**Space Complexity:** O(h) → Recursion stack (h = tree height)  

```cpp
class Solution {
public:
    bool isSymm(TreeNode* ptr1, TreeNode* ptr2) {
        if(!ptr1 && !ptr2) return true;
        if(!ptr1 || !ptr2) return false;
        return (ptr1->val == ptr2->val)
            && isSymm(ptr1->left, ptr2->right)
            && isSymm(ptr1->right, ptr2->left);
    }

    bool isSymmetric(TreeNode* root) {
        if(!root) return true;
        return isSymm(root->left, root->right);
    }
};
```

### ✅ Method 2: Iterative Approach (Using Queue / BFS)
Use a queue to iteratively check node pairs.

Push left and right nodes in pairs to the queue for comparison.

Continue until all pairs are validated or a mismatch is found.

**Time Complexity:** O(n) → Each node visited once  
**Space Complexity:** O(n) → Queue stores node pairs  

```cpp
class Solution {
public:
    bool isSymmetric(TreeNode* root) {
        if(!root) return true;
        queue<TreeNode*> q;
        q.push(root->left);
        q.push(root->right);

        while(!q.empty()) {
            TreeNode* leftNode = q.front(); q.pop();
            TreeNode* rightNode = q.front(); q.pop();

            if(!leftNode && !rightNode) continue;
            if(!leftNode || !rightNode) return false;
            if(leftNode->val != rightNode->val) return false;

            q.push(leftNode->left);
            q.push(rightNode->right);
            q.push(leftNode->right);
            q.push(rightNode->left);
        }
        return true;
    }
};
```

## 🛠️ Key Takeaways
- Recursion: Clean, intuitive, and easier to implement.
- Queue (Iterative): Avoids recursion stack overflow for very deep trees.
- Symmetric tree check is a classic mirror comparison problem.
