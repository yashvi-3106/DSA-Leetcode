# Delete Node in a Binary Search Tree (LeetCode 450) 📝

## Problem Statement

Given a **root node** of a **BST** and a value `key`, delete the node with the given key in the BST.  
Return the **root of the updated BST**.  

**Requirements:**  
- The deletion should maintain the properties of the BST.  
- If the node has **two children**, replace it with the **inorder successor** (smallest node in the right subtree).  

---

## Example

**Input:**  
root = [5,3,6,2,4,null,7], key = 3

**Output:**  
[5,4,6,2,null,null,7]

**Explanation:**  
- Node `3` has two children (`2` and `4`).  
- Replace `3` with its inorder successor `4`.  

---

## Approach

### Method 1: Recursion (Inorder Successor)

1. **Traverse the tree** to find the node with value `key`.  
2. **If node not found**, return `nullptr`.  
3. **If node found**:
   - **No left child:** Replace node with right child.  
   - **No right child:** Replace node with left child.  
   - **Two children:**  
     - Find the **inorder successor** (smallest node in the right subtree).  
     - Replace node’s value with successor’s value.  
     - Recursively delete the successor node in the right subtree.  

**Time Complexity:** O(h) → h is the height of the BST  
**Space Complexity:** O(h) → Recursion stack  

---

## Code (C++)

```cpp
/**
 * Definition for a binary tree node.
 * struct TreeNode {
 *     int val;
 *     TreeNode *left;
 *     TreeNode *right;
 *     TreeNode() : val(0), left(nullptr), right(nullptr) {}
 *     TreeNode(int x) : val(x), left(nullptr), right(nullptr) {}
 *     TreeNode(int x, TreeNode *left, TreeNode *right) : val(x), left(left), right(right) {}
 * };
 */

class Solution {
public:
    TreeNode* deleteNode(TreeNode* root, int key) {
        if (!root) return nullptr;

        if (root->val > key) {
            root->left = deleteNode(root->left, key);
        } else if (root->val < key) {
            root->right = deleteNode(root->right, key);
        } else {
            // Node to delete found
            if (!root->right) return root->left;
            if (!root->left) return root->right;

            // Node has two children
            TreeNode* tmp = root->right;
            while (tmp->left) tmp = tmp->left;  // inorder successor
            root->val = tmp->val;
            root->right = deleteNode(root->right, tmp->val);
        }

        return root;
    }
};
```

## Key Takeaways

- BST Deletion has three cases:
  - Node has no children → remove node.
  - Node has one child → replace node with child.
  - Node has two children → replace with inorder successor.
- Recursion simplifies tree traversal and deletion logic.
- Inorder successor ensures BST properties remain intact.
